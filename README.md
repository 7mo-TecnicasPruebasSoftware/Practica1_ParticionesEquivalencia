# Práctica 1: Particiones de Equivalencia con Pytest

Este proyecto implementa soluciones de software siguiendo la técnica de **Particiones de Equivalencia (PE)** para el diseño de pruebas unitarias. El objetivo es validar el comportamiento de funciones bajo rangos específicos y el manejo correcto de excepciones en Python.

## Estructura del Proyecto

practicas/
└── practica1/
    ├── src/
    │   ├── __init__.py
    │   ├── calcular_temperatura.py
    │   └── clasificar_calificacion.py
    └── tests/
        ├── __init__.py
        ├── test_calcular_temperatura.py
        └── test_clasificar_calificacion.py
